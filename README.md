🏦 Banking Program

One of my early Java projects — a simple banking app that runs in the terminal.A clean menu where you can deposit money, withdraw it, and check your balance.


What it does:

You get a menu with 4 options:


💰 Check your balance — see how much you have
➕ Deposit — add money to your account
➖ Withdraw — take money out (it won't let you go negative!)
🚪 Exit — close the program


It keeps running in a loop until you choose to exit. 


What I learned building this:


Breaking code into methods 
The difference between void methods and methods that return a value
Using switch expressions to handle menu choices cleanly
Validating user input so people can't deposit negative amounts or overdraft
